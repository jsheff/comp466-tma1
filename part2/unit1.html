HTML 5 Fundamentals

Introduction

HTML5 is a markup language, it describes the structure of a web page. HTML5 uses tags to label content which the browser renders to a page.

Here is a standard HTML5 document.

    <!DOCTYPE html>
    <html>

      <head>
        <title>My Web Page</title>
      </head>

      <body>
      </body>

    </html>
  
<!DOCTYPE html>

This the document type declaration. This tells the browser that this document is an HTML5 webpage that should be rendered in standards mode.

<html>

The html element tells the browser that everything that follows is HTML code.

<head>

The head element contains data about your web page. The head section can also contain document styling information such ass CSS3 style sheet or script references.

<title>

The title element describes the web page and is displayed at the top of the browser window or in the browser tab.

<body>

The body tag contains everything that should be displayed inside the main browser window.

<p id="my-paragraph">This is a paragraph<p>

An element consists of an opening tag, which is a left-angle bracket followed by the tag name and a right-angle bracket. The closing tag is a left-angle bracket, a forward slash, the tag name and a right-angle bracket. The opening and closing tag surround the tag's content. A tag may also contain attributes that provide additional information about the tag. An attribute consists of a name and a value separated by an equals sign.

Validation

An HTML validation service will ensure that the code you write is valid HTML. The W3C validator can check your document using:

  • the URL of your web page
  • the html file that you can upload
  • directly pasting in the code to a provided text area

Linking

<a href="http://google.com">Google</a>    Google
<a href="#top">Go to the top</a>>    Go to the top
<a href="mailto:joyce@email.com">Email joyce</a>    Email joyce

Links allow a user to move from one web page to another or from one part of a web page to a different part of the same web page.Any displayed element can act as a link. You can link to another web page or a location within a page, a file, or an email address. A link is created with an anchor element with an href attribute that specifies the location of the resource.

Images

<img src="image.jpg" alt="image description" width="100" height="100"/>

To add an image to the page, you use the img element. The src attribute tells the browser where the image is located. The alt attribute is used to describe the image when it cannot be rendered on the screen. You can specify the dimensions of the image using the attributes width and height. This can be useful so that the browser can render the text on the page while leaving space for the image. The img element is also an example of a void element. This tag does not need a closing tag because the img element does not surround content. Therefore you can close the tag by using a forward slash before the right-angle bracket.

Special Characters

Lists

An unordered list can be used to show items in a simple bullet-style list. Each entry in the list is a list item element <li>. Each item is rendered on a new line. An ordered list is used when you want to show items with each item numbered.
